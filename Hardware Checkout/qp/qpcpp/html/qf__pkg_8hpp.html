◆ QF_PTR_RANGE_

#define QF_PTR_RANGE_ (   x_,
  min_,
  max_ 
)    (((min_) <= (x_)) && ((x_) <= (max_)))

This macro is specifically and exclusively used for checking the range of a block pointer returned to the pool. Such a check must rely on the pointer arithmetic not compliant with the [AUTOSAR Rule M5-0-18]. Defining a specific macro for this purpose allows to selectively disable the warnings for this particular case.
