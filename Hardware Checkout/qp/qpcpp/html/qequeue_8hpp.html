Detailed Description

This header file must be included in all QF ports that use native QF event queue for active objects. Also, this file needs to be included in the QP/C++ library when the application uses QActive::defer() / QActive::recall(). Finally, this file is also needed when the "raw" thread-safe queues are used for communication between active objects and non-framework entities, such as ISRs, device drivers, or legacy code.
