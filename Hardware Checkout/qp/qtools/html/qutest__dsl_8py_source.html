179# This @ref qutest_simple "simple command" sends the complete

181# Any existing Global Filter setting inside the Target will be overwritten.
182#
183# @param[in] args list of Record-Type groups or individual Record-Types
184# to set or clear. A given filter-group or an individual filter is set when
185# it is positive, and cleared with it is preceded with the minus (`-`) sign.
186# <br>
187# The filter list can contain the following:
188# @code{.py}
189# GRP_ALL # all Record-Types

191# GRP_AO # Active Object Record-Types
192# GRP_MP # Memory Pool Record-Types
193# GRP_EQ # Event Queue Record-Types
194# GRP_TE # Time Events Record-Types
195# GRP_QF # Framework Record-Types (e.g., post/publish/..)
196# GRP_SC # Scheduler Record-Types (e.g., scheduler lock/unlock)
197# GRP_SEM # Semaphore Record-Types (e.g., Semaphore take/signal)
198# GRP_MTX # Mutex Record-Types (e.g., Mutex lock/unlock)
199# GRP_U0 # User group 0 (Record-Types 100-104)
200# GRP_U1 # User group 1 (Record-Types 105-109)
201# GRP_U2 # User group 2 (Record-Types 110-114)
202# GRP_U3 # User group 3 (Record-Types 115-119)
203# GRP_U4 # User group 0 (Record-Types 120-124)
204# GRP_UA # All user records (Record-Types 100-124)
205# <num> # Specific QS trace Record-Type in the range 0..127
206# @endcode
